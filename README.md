# Paradise Nursery 🌿

## Welcome to Paradise Nursery

Paradise Nursery is an online plant shop where you can find a wide variety of beautiful plants for your home and garden.

## Project Description

This is a React-based e-commerce application that allows users to:
- Browse a variety of houseplants
- Add plants to their shopping cart
- Update quantities and remove items from cart
- View total cost before checkout

## Technologies Used

- React.js
- Redux Toolkit
- Vite
- CSS3

## Getting Started

```bash
npm install
npm run dev
```
